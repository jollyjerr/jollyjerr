# Vim

Most important

```
:w save
:q quit
:q! quit without saving
u undo
ctrl R redo
:help help
```

### Modes

```
i,o,O,etc... insert mode
esc normal mode
v visual mode
ctrl-V visual block
```

### Movement

`hjkl => move by letter , wbe => move by word`

```
a end of word
A end of line
number action (repeat x times)
f,t jump in line
% jump brackets
0 full left
$ full right
G bottom
gg top
:number line number
ctrl-D jump down
ctrl-U jump up
/search-term, n || N
```

### Editing

```
dtl 'delete to l'
y,yy copy
. repeat last command
```
